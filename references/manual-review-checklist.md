# Manual review checklist

- [ ] Публичная версия не содержит chat IDs, user IDs и private repo refs
- [ ] Текст написан понятным русским, а не сухим техно-канцеляритом
- [ ] Человеку понятно, что делать первым сообщением
- [ ] Правило "один topic = один workstream" явно зафиксировано
- [ ] В docs отражён фикс `inlineButtons = "all"`
- [ ] В docs отражён footgun с `bindings` array overwrite
- [ ] Инструкция не врёт о том, что после bind можно писать обычным текстом
